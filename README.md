# 智聘 Copilot — AI 招聘决策助手

面向中小企业的一站式 AI 招聘系统，让招聘更高效、更精准。

## 核心功能

### 1. AI 生成岗位 JD
输入岗位基础信息，AI 自动生成结构化的岗位描述，包括职责、要求、薪资建议，并自动生成匹配规则（硬性条件、加分项、淘汰项、面试重点）。

### 2. 简历批量导入与解析
支持 PDF/DOCX/TXT 格式简历拖拽上传，AI 自动解析姓名、学历、技能、工作经历、薪资期望等关键信息。

### 3. 可解释候选人评分引擎
基于证据链的多维度评分系统：
- 6 个评分维度：硬性匹配、技能匹配、项目相关性、薪资匹配、稳定性、潜力
- 每个分数必须有简历原文作为证据依据
- HR 可自定义各维度权重，实时重算总分

### 4. 招聘 Copilot 今日待办
智能工作台自动识别待办事项：
- 新投递候选人待筛选
- 超过 24 小时未反馈的面试
- 高匹配但未联系的候选人
- Offer 风险候选人跟进
- 招聘 ROI 数据统计（节省工时、成本、周期）

### 5. 面试闭环管理
完整的面试生命周期：
- **准备阶段**：AI 生成个性化面试问题，每题附带评分标准和验证方式
- **面试记录**：面试官逐题打分（1-5 分），记录候选人回答要点
- **反馈汇总**：自动汇总评分、生成综合评语、给出录用建议

### 6. 飞书集成
支持飞书群机器人 Webhook 真实推送：
- JD、面试题、评分报告一键推送到飞书群
- 飞书消息卡片预览
- Webhook 连通性测试

## 技术栈

- **框架**：Next.js 16 + React 19
- **样式**：Tailwind CSS 4
- **语言**：TypeScript
- **部署**：支持 Vercel 一键部署

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

访问 http://localhost:3000 即可使用。

## 页面结构

| 路径 | 功能 |
|------|------|
| `/` | 招聘工作台：今日待办 + 招聘漏斗 + ROI 数据 |
| `/create-job` | 创建岗位：AI 生成 JD + 匹配规则 |
| `/candidates` | 简历筛选：证据链评分 + 简历上传 |
| `/interview` | 面试管理：问题生成 → 面试记录 → 反馈汇总 |
| `/report` | 评分报告：雷达图 + 多维度评分 |
| `/compare` | 候选人对比：横向对比 + AI 综合推荐 |
| `/feishu` | 飞书集成设置 |
